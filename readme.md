# 总结
========================================================================================

  数据类型                数字类型和非数字类型
  数字类型                整型，浮点型，长整型，复数型
  非数字类型              字符串，列表，元组，字典
  非数字类型的共同点       可以使用切片，链接（+），重复（*），取值等相关云散
  非数字类型的不同点       列表可以直接赋值，元组不可以赋值，字典按照k-v赋值

   ps: isinstance 会认为子类是父类的类型
       type 不会认为 子类是父类的类型

=========================================================================================

一/运算符
    ** 指数运算符
    is / is not   身份运算符
    in / not in   成员运算符
    and/or/not    逻辑运算符
